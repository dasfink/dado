# Common Shed Plans Reference

## Standard Shed Sizes

| Size | Use | Foundation |
|------|-----|-----------|
| 8x8 | Garden tools, small storage | On-grade blocks |
| 8x10 | Workshop, storage | On-grade blocks |
| 8x12 | Workshop, sauna conversion | On-grade blocks |
| 10x12 | Large workshop | Concrete pier or slab |
| 10x16 | Garage alternative | Concrete slab |
| 12x16 | Two-car storage, workshop | Concrete slab |
| 12x20 | Large workshop/studio | Concrete slab |

## 8x12 Shed (Most Common for Woodworking)

### Floor Framing
- 2x6 PT joists, 16" OC
- 6 joists + 2 rim joists = 8 pieces at 12'
- 3/4" CDX plywood subfloor: 3 sheets (4x8)

### Wall Framing (8' walls)
- Front wall: 12' sole plate, 12' double top plate, ~9 studs 16" OC, door header
- Back wall: 12' sole plate, 12' double top plate, ~9 studs 16" OC
- Side walls: 8' sole plate, 8' double top plate, ~6 studs 16" OC each
- Total studs: ~30 (plus cripples and extras)

### Roof Options

**Gable roof:**
- Ridge board: 2x8 or 2x10 at 12' + overhangs
- Rafters: 2x4 or 2x6, ~16 pairs at 16" OC
- Collar ties: 2x4 every 4' (or every other rafter pair)
- Sheathing: 7/16" OSB

**Shed roof (single slope):**
- Front wall height: 8' (96")
- Back wall height: 6'8" (80") for 2/12 pitch, or 7'4" (88") for 1/12
- Rafters: 2x6 at 16" OC, ~10 rafters
- Simpler construction, good for lean-tos

### Door and Window
- Standard door: 3'0" x 6'8" (36" x 80")
- Rough opening: 38" x 82"
- Window: 24" x 36" or 30" x 36"

## Lean-To Structure

A lean-to attaches to an existing building:

### Key Dimensions
- Width (projection from wall): 4' to 8' typical
- Length: matches existing wall or portion
- Roof attaches to existing wall via ledger board
- Minimum 1/4" per foot slope away from building

### Framing
- **Ledger board:** 2x6 or 2x8, lag-bolted to existing wall (3/8" x 4" lag screws, 16" OC)
- **Posts:** 4x4 at front, 6' to 8' OC
- **Beam:** 2x6 or 2x8 doubled, on top of posts
- **Rafters:** 2x4 or 2x6, from ledger to beam, 16" or 24" OC
- **Knee braces:** 2x4 at 45° from post to beam (if no sheathing)

### Foundation
- Concrete piers (Sonotubes) at each post location
- Depth: below frost line (varies by region, 36"-48" in northern US)
- Diameter: 8" to 12"
- Post anchors: Simpson Strong-Tie AB or equivalent
